Gallery Vorlage | Codrops

Die Gallery Vorlage ist von Codrops - mit geringfügige Änderungen, um sie unter Django zu implementieren. Sie ist im Netz unter "Responsive Image Gallery" (A JQuery image gallery with a thumbnail carousel) zu finden. Mir gefiel das Konzept, für jede Fotoserie eine eigene Gallerie zu erstellen. https://tympanus.net/codrops/. Auf der Seite wird ausführlich in einem Tutorial gezeigt, wie der Quellcode angepasst werden kann.

Django Tutorials

Leider habe ich kaum aktuelle Tutorials auf deutsch gefunden, wenn überhaupt. Ohne englisch kommt man hier kaum weit. Ähnlich sieht es bei gedruckten Werken aus - die Bücher sind vorwiegend auf englisch. Die Webseite von Vito Freitas ist sehr empfehlenswert (simpleisbetterthancomplex). Er hat ein 7-teiliges Tutorial gemacht, das unheimlich viele Aspekte von Django abdeckt. Zudem kann er sehr gut erklären. Für absolute Anfänger empfielt sich die Seite "Djangogirls" - und eben immer wieder der Blick in die Django Documentation.
